## 💡 解决方案

### 方案 A：配置本地 JAR（推荐）

**步骤：**

1. **在 pom.xml 中添加本地 JAR 配置**
   ```xml
   <dependency>
       <groupId>com.hyplat</groupId>
       <artifactId>alipay-sdk</artifactId>
       <version>1.0.0</version>
       <scope>system</scope>
       <systemPath>${project.basedir}/hyplatlib/alipay-sdk.jar</systemPath>
   </dependency>
   ```

2. **配置所有 hyplatlib 中的 JAR**
   - alipay-sdk-*.jar
   - junziqian-sdk.jar
   - mqtt-client.jar
   - jxl.jar
   - 等等（约 50 个）

**预计时间：** 2-3 小时

---

### 方案 B：使用 Ant 编译（原有方式）

**步骤：**

   - 已更新路径为 Linux 兼容路径

2. **执行编译**
   ```bash
   ant build
   ```

**前提：** 需要安装 Ant
